# Forge & Pixel Tools

Trois mini-outils web prêts à l'emploi. Un seul fichier HTML par outil, zéro dépendance, zéro compilation, déploiement immédiat.

🌐 **Démos live** : https://forge-pixel-tools.pages.dev/

## Le catalogue

### [HashCraft](./hashcraft/) — Générateur de hashtags
Génère des hashtags ciblés par plateforme (Instagram, TikTok, LinkedIn, X, YouTube). Détection auto du sujet, export CSV.

### [InvoiceForge](./invoiceforge/) — Générateur de factures PDF
Crée une facture pro et télécharge le PDF en 2 minutes. Multi-devises (EUR, USD, GBP, CHF, CAD), TVA auto.

### [QR Pixel](./qrpixel/) — Générateur de QR codes hors-ligne
Génère des QR codes 100% local. 6 formats (URL, texte, email, tél, WiFi, vCard), personnalisation couleur/forme, export PNG + SVG.

## Utilisation

Chaque outil est un fichier `index.html` autonome. Pour l'utiliser :

```bash
# Ouvrir directement dans un navigateur
open hashcraft/index.html
```

Ou déployer sur n'importe quel hébergeur statique (Cloudflare Pages, Netlify, Vercel, GitHub Pages…).
