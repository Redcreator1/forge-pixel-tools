# HashCraft

Générateur de hashtags multi-plateformes.

## Utilisation

Ouvrir `index.html` dans un navigateur. Aucune installation, aucune dépendance.

## Fonctionnalités

- 5 plateformes : Instagram, TikTok, LinkedIn, X, YouTube
- Détection automatique du sujet du contenu
- Export CSV et copie en un clic
