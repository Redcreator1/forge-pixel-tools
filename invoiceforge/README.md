# InvoiceForge

Générateur de factures PDF.

## Utilisation

Ouvrir `index.html` dans un navigateur. Aucune installation, aucune dépendance.

## Fonctionnalités

- Multi-devises : EUR, USD, GBP, CHF, CAD
- Calcul automatique des sous-totaux et de la TVA
- Aperçu live + export PDF en un clic
