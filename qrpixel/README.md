# QR Pixel

Générateur de QR codes 100% hors-ligne.

## Utilisation

Ouvrir `index.html` dans un navigateur. Aucune installation, aucune dépendance, aucune donnée envoyée à un serveur.

## Fonctionnalités

- 6 types : URL, texte, email, téléphone, WiFi, vCard
- Couleurs, formes et taille personnalisables
- Export PNG + SVG
